## Overview
**Paralives** is a vibrant life simulation game where you create unique characters (Parafolks), build dream homes with full creative freedom, and manage their lives in an open-world town.

This repository provides the **Early Access Release** for PC.

> [!Important]
> Paralives Early Access launched on **May 25, 2026**.

## Key Features
- Highly detailed Paramaker (Character Creator)
- Gridless building system with curved walls and split-levels
- Deep character customization and genetics
- Open world town exploration
- Rich life simulation mechanics (emotions, careers, relationships)
- Beautiful graphics and smooth gameplay
- Regular free updates during Early Access

## System Requirements
- **OS**: Windows 10 / Windows 11
- **Processor**: Modern multi-core CPU
- **Memory**: 16 GB RAM recommended
- **Graphics**: Dedicated GPU with 6GB+ VRAM
- **Storage**: ~25 GB available space
